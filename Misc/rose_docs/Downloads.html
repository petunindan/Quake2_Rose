To download, click on the link below. Please note. This release requires the 3.20 version of quake2. Please update your quake2 before installing Window Of Retaliation.

To install.

1) Download the zip file below

2) Unzip onto your hard drive

3) Run setup from there

4) Ensure you enter the correct path for your existing quake2 directory when prompted. Ie dont try adding a sub directory

 

If you get stuck when playing the game, check out the "hints" section of the readme file.
